Introduction

Python is a very popular programming language, first released in 1991. It is seen as a good language to start learning programming with, due to it being highly readable

Python works on multiple different platforms, and can be used for many tasks, such as:

  • back-end web development
  • software development
  • system scripting

For this documentation, all work will be done in the Python IDLE that comes bundled with the Python intallation.

Hello World

Python is an interpreted language, which means that as the developer, you write in Python (.py) files and then put those files into the python interpreter to be executed

To create a python file that prints the words "Hello World", we can write the following in our .py file:

                print("Hello World")

Now select "Run > Run Module" in the toolbar, or press F5 to run the program. The output should read:

                Hellow World

Congratulations, you have just written your first Python program.

Variables

Variables are containers for storing data values.

Unlike other programming languages, Python has no command for declaring variables.

Variables are just created the moment you first assign a value to it.

There are certain rules for naming variables in Python. They are:

  • Variable names must start with either a letter or the underscore (_) character
  • Variable names cannot start with a number
  • Variable names can only contain alpha-numeric characters and underscores
  • Variable names are case sensitive
Data Types

Python comes with a few data types built-in by default. They are:

  • Text (str)
  • Numbers (int, float, complex)
  • Sequences (list, tuple, range)
  • Mappings (dict)
  • Sets (set, frozenset)
  • Boolean values (bool)
  • Binary values (bytes, bytearray, memoryview)

Using these data types, we can create very powerful functions with our programs.

if else Statement

Use the if statement to execute a statement if the logical condition is true.

                if condition:
                    statement

An optional else statement can also be used to execute if the logical condition is false.

                if condition:
                    statement_when_true
                else:
                    statement_when_false

Else if statements can also be used between the if and else statements to say "if the previous conditions were not true, then try this condition".

                if condition_1:
                    statement_when_condition1_true
                elif condition_2:
                    statement_when_condition1_false_condition2_true
                else:
                    statement_when_condition1_false_condition2_false
while Loops

If you need to execute a block of code multiple times, a while loop can come in very handy.

A while loop will execute a block of code as long as a condition is true.

                while condition:
                    code

It is usually a good idea to have an exit condition in the code inside the while block as otherwise the block of code will run infinitely.

The break statement can stop and exit out of the loop, even if the condition is still true.
